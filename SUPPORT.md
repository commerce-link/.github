# Support

## Getting Help

### GitHub Discussions

For questions, ideas, and general discussion, use [GitHub Discussions](https://github.com/orgs/commerce-link/discussions).

### Issue Tracker

Found a bug? Use the issue tracker in the relevant repository with the appropriate issue template:

- [Bug Report](https://github.com/commerce-link/.github/issues/new?template=bug-report.yml)
- [Feature Request](https://github.com/commerce-link/.github/issues/new?template=feature-request.yml)
- [Integration](https://github.com/commerce-link/.github/issues/new?template=integration.yml)

### Documentation

Each repository has its own `README.md` with setup instructions and configuration details.

### Security Issues

**Do not open public issues for security vulnerabilities.** See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
